# Daily Math Problem Template

### Problem Metadata
Core information about the problem:
- **Date**: Problem assignment date (YYYY-MM-DD)
- **Problem Number**: Sequential identifier
- **Topic**: Mathematical area (e.g., Matrix Determinants, Eigenvalue Decomposition)
- **Category**: Machine Learning Math / Deep Learning Math / AI Math & Theory / Advanced Mathematics
- **Difficulty Level**: Beginner / Intermediate / Advanced
- **Estimated Time**: Expected completion time
- **Prerequisites**: Required background knowledge

### Problem Statement
Complete problem description including all constraints, conditions, and objectives.

### Source and Attribution
- **Primary Source**: Original reference (textbook, paper, problem set)
- **Related Materials**: Supplementary references

### Motivation and Context
- Why the problem matters
- Relevance to ML/DL/AI applications
- Theoretical significance and concepts illustrated

### Hints and Guidance
Strategic hints without revealing solutions:
- Relevant theorems or techniques
- Structural observations
- Possible approaches

### Discussion Space
Collaborative areas for:
- Clarifying questions
- Initial observations and insights

### Status Tracking
- **In Progress**: List of participants currently working on the problem

## How to Use

1. **Create a new problem page** using this template
2. **Fill in metadata** to categorize and contextualize the problem
3. **State the problem** clearly with all necessary information
4. **Add hints** to guide without spoiling the solution
5. **Collaborate** using the discussion space
6. **Submit solutions** with links to detailed writeups
7. **Share resources** that aided understanding

> ### Categories Explained    
> - **Machine Learning Math**: Linear algebra, optimization, probability    
> - **Deep Learning Math**: Backpropagation, activation functions, loss functions    
> - **AI Math & Theory**: Computational complexity, information theory     
> - **Advanced Mathematics**: Abstract algebra, real analysis, topology
